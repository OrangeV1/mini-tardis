* sparks (and associated sounds) are created with `createInteriorSparks()` in `minitardis.component.Tardis`
* `minitardis.component.Tardis` is the main class for tardises (tardi?)
* to add new interiors, there needs to be an nbt file in `resources/data/mini_tardis/structure/interiors` and a definition file in `resources/data/mini_tardis/mini_tardis/interiors`
